# 🎟️ Eventra – Event Management & Ticketing Platform

Eventra is a full-stack event management and ticketing web application designed to provide a seamless experience for customers and event organizers.

The platform allows customers to discover events, purchase tickets, and manage their digital tickets, while organizers can create and manage events through a dedicated organizer dashboard.

## ✨ Features

### 👤 Customer Features
- User registration and login
- Browse and explore events
- View event details
- Purchase event tickets
- View purchased tickets in My Tickets
- Digital ticket details
- Protected user routes

### 🎪 Organizer Features
- Organizer dashboard
- Create new events
- Edit existing events
- Delete events
- Upload event images
- Manage event information
- Dashboard statistics

### 🚧 Planned Features
- Sales analytics dashboard
- QR code ticket generation
- QR ticket scanning and validation
- Additional organizer management features

## 🚀 Project Status

Eventra is currently under active development.

Customer authentication, ticket purchasing, digital tickets, organizer event creation, event editing, event deletion, and image uploading have been implemented.

Current development focuses on integrating uploaded event images with event creation and displaying organizer-created events on the customer Explore page.
